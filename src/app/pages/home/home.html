Web API Endpoint Reference

Start here to learn the essentials about integration with mailapp.



POST http://www.api.mailapp.pl

Endpoint

This endpoint give external applications possibility to add email (not only) to your subscriber group.

{
  "email": "thomas@contact.com",
  "group": "GROUP_KEY",
  "attributes": [
      {
        "name": "name",
        "value": "Thomas"
      },
      {
        "name": "gender",
        "value": "man"
      }
  ]
}

Request Body

Name Required Description
email YES The email address coming from the field on the subscribe form.
group YES The group key generated when you create new group in the admin panel.
attributes NO Unlimited additional attributes consisting of name and value pairs. 
{
  "status": "success",
  "message": "The email address has been saved correctly"
  "code": 100
}

Response Format

On success, the HTTP status code in the response header is 201 Created and the response body contains status "success".

Example 1. Basic subscribe form

<form class="form-inline" id="subscribe-form">
  <div class="form-group">
    <label for="subscribe-email">Subscribe to our newsletter:</label>
    <input type="email" id="subscribe-email" class="form-control"/>
  </div>
  <button type="submit" class="btn btn-primary">Subscribe</button>
</form>

$( "#subscribe-form" ).submit(function( event ) {
  event.preventDefault();
  var data = {
    email: $( "#subscribe-email" ).val(),
    group: "GROUP_KEY"
  };

  $.post( http://www.api.mailapp.pl, JSON.stringify(data))
    .done(function( response ) {
      alert.("Email been saved correctly");
    })
    .fail(function( response ) {
      alert.("Something has gone wrong...");
    });
});
